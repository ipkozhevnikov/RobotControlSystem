# Управление мобильным роботом

Проект представляет собой систему управления мобильным роботом в симуляторе **IoTRobotWorld**.

Робот автоматически сортирует заготовки по цвету, перемещая их между платформами с использованием датчиков и заданного алгоритма движения.

---

## 📌 Функционал

- **Подключение к роботу** — установка UDP-соединения с симулятором.
- **Автономное движение** — перемещение по рабочему полю с использованием 8 лазерных дальномеров и смарт-камеры.
- **Навигация по секторам** — последовательное движение от сектора 1 до 3 без возврата в пройденные зоны.
- **Захват заготовок** — распознавание типа детали (красная / синяя) с помощью камеры и смарт-подхват.
- **Сортировка** — размещение заготовок на соответствующих платформах (красные → платформа 2, синие → платформа 3).
- **Корректировка траектории** — автоматическое выравнивание движения при отклонении от курса (каждую 1 секунду).
- **Обработка столкновений** — остановка движения при срабатывании контактного бампера.
- **Логирование** — вывод хода работы и ошибок в текстовое окно.

---

## ▶️ Запуск проекта

1. Запустите симулятор (приложение-клиент) из папки **симулятор_роботов** - файл **IoTRobotWorld.exe**
2. Запустите приложение-сервер (управляющее приложение)
3. Перед запуском убедитесь, что порты и IP адреса в обоих приложениях указаны верно  
4. В симуляторе нажмите кнопку подключения, затем запустите симуляцию Show Simulator Viewport -> Start Simulation
5. В серверном приложении также нажмите кнопку подключения, затем запустите алгоритм кнопкой Старт
6. Дождаться разрешающего сигнала светофора  
7. Робот начнёт выполнение задачи автоматически  

---

## 🔄 Логика работы

Робот выполняет следующий цикл:

1. Ожидание сигнала старта  
2. Движение к платформе  
3. Определение цвета объекта  
4. Захват  
5. Доставка:
   - красный → платформа №2  
   - синий → платформа №3  
6. Повтор цикла  

После завершения — переход в финишную зону
